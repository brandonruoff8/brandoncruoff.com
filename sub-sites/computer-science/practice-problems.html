Practice Problems

Two Sum Problem

This program will take in a single number and a list of numbers, and will figure out which pairs
of numbers from the list will add up to the single number.



Integer to Roman Numeral

This program will take a single number and turn it into roman numerals.


Palindromic Substring

This program will take in a single word and find the largest palindrome (word that is
the same forwards and backwards) lies within the given word. If the largest is of size 1
(a single letter) then it returns the first letter in the word.
